# Forum Post

Website aplikasi forum sederhana.

## Fitur

- ✅ Authentication (Login/Register/Logout)
- ✅ Buat post dengan teks dan gambar
- ✅ Kategori post
- ✅ Like dan komentar
- ✅ Real-time updates dengan Socket.io
- ✅ Search/pencarian post
- ✅ Responsive design (mobile & desktop)

## Tech Stack

### Frontend

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- Socket.io Client
- React Router
- Axios

### Backend

- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Socket.io
- Multer (file upload)
